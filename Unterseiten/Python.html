Python

Was ist Python?

Python ist eine einfach zu lernende, vielseitige Programmiersprache. Sie wird in Webentwicklung, Datenanalyse, KI und Automatisierung eingesetzt.

Einfaches Beispiel: FizzBuzz

Das folgende Programm prüft Zahlen von 0 bis 100 auf Teilbarkeit durch 3 und 5:

def nummer():
    x = 0
    while x < 101:
        if x % 3 == 0 and x % 5 == 0:
            print(x, ": fizzbuzz")
        elif x % 3 == 0:
            print(x, ": fizz")
        elif x % 5 == 0:
            print(x, ": buzz")
        else:
            print(x, ": nix")
        x += 1

Arbeiten mit Bibliotheken: Matplotlib & NumPy

Das folgende Beispiel zeigt, wie man mit Zufallszahlen arbeitet und ein Diagramm erzeugt:

import random
import matplotlib.pyplot as plt
import numpy as np

def number(n, m):
    a = b = c = d = e = 0
    o = 0

    while o < n:
        x = random.randrange(1, m)
        if x == 1:
            a += 1
        elif x == 2:
            b += 1
        elif x == 3:
            c += 1
        elif x == 4:
            d += 1
        elif x == 5:
            e += 1
        o += 1

    print("a:", a, "b:", b, "c:", c, "d:", d, "e:", e)

    xpoints = np.array([0, "a", "b", "c", "d", "e"])
    ypoints = np.array([0, a, b, c, d, e])
    plt.plot(xpoints, ypoints)
    plt.show()
