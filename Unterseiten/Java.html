Java; Grundlagen: Klassen, Syntax...

Was ist Java?

Java ist eine objektorientierte Programmiersprache, die plattformunabhängig funktioniert. Sie ist weit verbreitet in der Softwareentwicklung, z. B. bei Web-Anwendungen, Android-Apps oder Unternehmenssoftware.

Grundlegende Datentypen in Java

Java bietet primitive Datentypen wie int (Ganzzahlen), double (Fließkommazahlen), char (einzelne Zeichen) und boolean (true/false). Zusätzlich gibt es Klassen wie String für Texte.

Beispiel:

int number = 5;
double number2 = 1.2;
char char1 = 'A';
String test = "A, B, C, D, E, F";

Mathematische Operationen

Java stellt viele mathematische Funktionen bereit, etwa mit der Klasse Math. Man kann Zufallszahlen erzeugen und runden:

double result = Math.ceil(Math.random() * 500) / 100;

Benutzereingaben mit Scanner

Mit der Klasse Scanner lassen sich Eingaben über die Konsole verarbeiten:

Scanner x = new Scanner(System.in);
System.out.print("Username: ");
String username = x.nextLine();

Verzweigungen mit if/else

Entscheidungen werden mit if/else-Anweisungen getroffen:

if ("Ja".equalsIgnoreCase(save)) {
    System.out.println("Top!");
} else if ("Nein".equalsIgnoreCase(save)) {
    System.out.println("Error 404; restart!");
}

Einfache Taschenrechner-Logik

Mit Bedingungen und Operatoren kann ein einfacher Rechner umgesetzt werden:

if (operator.equals("+")) {
    System.out.println(ersterTeil + zweiterTeil);
} else if (operator.equals("*")) {
    System.out.println(ersterTeil * zweiterTeil);
}

So lassen sich Texteingaben in Zahlen umwandeln und mit verschiedenen Operatoren verarbeiten.
